## On-Page SEO Standards

### Title Tag Formula
```
[Primary Keyword] - [Benefit] | LLCPad
```
- Keep between 50-60 characters
- Front-load primary keyword
- Include brand name at end

### Meta Description Formula
```
[Action verb] + [Service] + [Key benefit] + [Trust element] + [CTA]
```
- Keep between 150-160 characters
- Include primary keyword naturally
- Add urgency or unique value proposition

### URL Structure
```
/services/[service-slug]
```
- Use lowercase, hyphenated slugs
- Keep URLs short and descriptive

### Heading Hierarchy
- **H1:** Service name with primary keyword (one per page)
- **H2:** Major sections (Benefits, Process, Pricing, FAQs)
- **H3:** Subsections within H2s

### Image Optimization
- File names: `llc-formation-service.jpg`
- Alt text: Descriptive, keyword-rich
- Compress for fast loading (<100KB)

## Content Structure Template

Every service page should follow this structure:

### 1. Hero Section
- **Headline (H1):** Service name with benefit
- **Subheadline:** 1-2 sentences explaining value proposition
- **Trust badges:** "10,000+ Businesses Formed" | "4.9/5 Rating" | "24-48 Hour Processing"
- **Primary CTA:** "Get Started - From $XX"
- **Secondary CTA:** "Talk to an Expert"

### 2. Pain Points Section
- Address 3-4 specific problems your target audience faces
- Use empathetic language ("We understand...")
- Show you know their challenges

### 3. Solution Overview
- How your service solves their problems
- Key differentiators from competitors
- Unique value proposition

### 4. What's Included (Features)
- Bullet points with checkmarks
- Group into categories if many features
- Highlight most valuable items

### 5. Process Steps
- 3-5 clear steps
- Timeline expectations
- What customer needs to provide

### 6. Pricing Packages
- Clear package comparison
- Highlight "Most Popular" option
- Show value at each tier

### 7. Trust & Social Proof
- Customer testimonials (with photos/names)
- Success statistics
- Partner/certification logos
- Money-back guarantee

### 8. FAQs
- 5-8 questions minimum
- Target long-tail keywords
- Use schema markup

### 9. Related Services
- Cross-sell complementary services
- "Customers also bought" style

### 10. Final CTA
- Repeat main offer
- Urgency element if applicable
- Contact option for questions
