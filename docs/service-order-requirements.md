# Service Order Requirements Documentation

This document outlines the required information for each service to properly fulfill customer orders.

---

## 1. LLC Formation (Already Implemented)

### Required Fields:
- LLC Name (+ 2 alternative names)
- Formation State
- LLC Type (Single/Multi-member)
- Management Structure (Member/Manager-managed)
- Business Purpose/Industry
- Owner Information:
  - Full Name, Email, Phone
  - Country of Residence, City, Address
  - Passport Number (for international)
  - Ownership Percentage
- Additional Members (for Multi-member):
  - Full Name, Email, Phone
  - Country, City, Address
  - Ownership Percentage
- Non-Member Manager (if applicable):
  - Full Name, Email, Phone, Address, City, Country
- Expedited Processing Option

## 2. EIN Application

### Required Fields:
- **LLC Information:**
  - LLC Name (legal name)
  - State of Formation
  - Date of Formation
  - Physical Address (street, city, state, zip)
  - Mailing Address (if different)

- **Business Details:**
  - Type of Entity (LLC, Corporation, etc.)
  - Principal Business Activity/Industry
  - Business Start Date
  - Expected Number of Employees
  - First Date Wages Paid (if applicable)

- **Responsible Party Information:**
  - Full Legal Name
  - SSN or ITIN (or "Foreign" if none)
  - Title/Position
  - Address
  - Phone Number

- **Reason for Applying:**
  - Started new business
  - Banking purposes
  - Changed type of organization
  - Compliance

### FAQs:
1. Do I need an SSN to get an EIN? - No, international owners can get EIN without SSN/ITIN
2. How long does it take? - 4-6 weeks for international applicants
3. What if my LLC isn't formed yet? - You need to form the LLC first
4. Can I use EIN for Amazon? - Yes, EIN is required for Amazon seller account

---

## 3. ITIN Application

### Required Fields:
- **Applicant Information:**
  - Full Legal Name (as on passport)
  - Date of Birth
  - Country of Birth
  - Country of Citizenship
  - Current Foreign Address
  - US Address (if applicable)

- **Passport Information:**
  - Passport Number
  - Passport Issue Date
  - Passport Expiration Date
  - Country of Issue

- **Tax Filing Reason:**
  - Filing Tax Return
  - Claiming Tax Treaty Benefits
  - Third-Party Withholding
  - Other (specify)

- **US Income Information:**
  - Type of Income
  - Tax Year
  - Payer Name (if applicable)

- **Certification Option:**
  - Standard (mail original passport)
  - CAA Service (no original needed)

### FAQs:
1. What documents do I need? - Valid passport is primary, or 2 alternative documents
2. Do I need to mail my original passport? - Not with CAA certification
3. How long does it take? - 7-11 weeks processing time
4. Does ITIN expire? - Yes, if not used for 3 consecutive years

## 4. Trademark Registration

### Required Fields:
- **Trademark Information:**
  - Trademark Type (Word Mark / Design Mark / Combined)
  - Trademark Text (exact spelling)
  - Logo/Design Upload (if applicable)
  - Color Claim (yes/no, specify colors)

- **Owner Information:**
  - Owner Type (Individual / Business Entity)
  - Legal Name
  - Address (street, city, state/country, zip)
  - Citizenship/State of Incorporation
  - Email and Phone

- **Goods/Services:**
  - Description of Goods/Services
  - International Class(es) (1-45)
  - Specimen Upload (photo showing trademark in use)

- **Filing Basis:**
  - Use in Commerce (already using)
    - Date of First Use Anywhere
    - Date of First Use in Commerce
  - Intent to Use (not yet using)

- **Additional Options:**
  - Expedited Examination (additional fee)
  - Trademark Search Before Filing

### FAQs:
1. How long does registration take? - 8-12 months on average
2. Do I need a trademark for Amazon Brand Registry? - Yes
3. What's the difference between TM and R symbol? - R is for registered marks only
4. Can I trademark a name already in use? - Only if not confusingly similar

- **Filing Location:**

- **Owner Information:**
  - Primary Contact Name

### FAQs:
1. What's the difference between DBA and LLC? - LLC is legal entity, DBA is just alternate name
2. Can I have multiple DBAs? - Yes, you can register multiple DBAs
3. Does DBA protect my name? - No, it provides limited protection
4. Do I need a DBA for each state? - Often yes, if operating under DBA in that state

## 6. Operating Agreement

### Required Fields:
- **LLC Information:**
  - LLC Name
  - State of Formation
  - Date of Formation
  - Principal Business Address
  - Business Purpose

- **Member Information:**
  - Member Names
  - Member Addresses
  - Ownership Percentages (must total 100%)
  - Capital Contributions

- **Management Structure:**
  - Member-Managed or Manager-Managed
  - Manager Name(s) if Manager-Managed
  - Voting Rights Allocation

- **Financial Provisions:**
  - Profit Distribution Method (proportional/equal/custom)
  - Loss Allocation
  - Distribution Timing (monthly/quarterly/annually/as decided)

- **Additional Provisions (Optional):**
  - Buy-Sell Provisions
  - Transfer Restrictions
  - Dissolution Procedures
  - Non-Compete Clauses

### FAQs:
1. Is operating agreement required? - Required in 6 states, recommended in all
2. Does it need to be notarized? - No, but all members should sign
3. Can I change it later? - Yes, with member consent
4. Do I file it with the state? - No, it's an internal document
